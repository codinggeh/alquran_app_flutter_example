# Al-Quran App Demo

This is a demo Flutter web application showcasing mobile app development skills.

## Live Demo

Visit the live demo: [https://coding-geh.github.io/alquran_app/](https://coding-geh.github.io/alquran_app/)

## Features

- 📱 Modern Flutter UI/UX Design
- 🌐 Web Deployment
- 📚 Quran Surah List
- 🔖 Bookmark System
- 🎵 Audio Playback
- 🌍 Multi-language Support
- 📱 Responsive Design

## Tech Stack

- **Framework**: Flutter 3.32.2
- **Language**: Dart
- **State Management**: Provider
- **Localization**: easy_localization
- **Deployment**: GitHub Pages
